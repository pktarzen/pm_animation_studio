# PM Blender Tools

Free and open-source Blender add-ons and AI-powered tools for character creation, animation, and game development.

## Overview

PM Blender Tools is an open-source project focused on simplifying 3D character creation and Blender workflows. The goal is to provide artists, indie game developers, and content creators with free, easy-to-use tools that improve productivity.

This project is actively developed and welcomes community feedback and contributions.

## Features

* Blender Add-ons
* Character Workflow Automation
* AI-Assisted Character Generation
* Outfit and Asset Management
* Blender Utility Scripts
* Open Source and Free to Use

## Planned Features

* Batch Processing Tools
* AI Integration for Blender

## Installation

1. Download the latest release.
2. Open Blender.
3. Go to **Edit → Preferences → Add-ons**.
4. Click **Install**.
5. Select the downloaded ZIP file.
6. Enable the add-on.

## Requirements

* Blender 4.x or later
* Windows, Linux, or macOS
