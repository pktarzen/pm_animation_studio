# PM Blender Tools

Free and open-source Blender add-ons and AI-powered tools for character creation, clothing workflows, animation, and game development.

## Overview

PM Blender Tools is an open-source collection of Blender add-ons designed to simplify character creation and automate repetitive tasks. The project focuses on practical tools for artists, students, indie game developers, and content creators.

The first add-on in this project is **PM Auto Outfit Fitter**, which helps fit clothing meshes to character body meshes with minimal manual work.

This project is actively developed, and new tools are added regularly.

---

## Current Add-ons

### PM Auto Outfit Fitter

Current features:

* Analyze character body dimensions
* Automatically resize clothing to match the body
* Simple Blender Sidebar UI
* Works with standard Blender mesh objects
* Lightweight and open source

## Installation

1. Download the latest Release.
2. Open Blender.
3. Go to **Edit → Preferences → Add-ons**.
4. Click **Install from Disk**.
5. Select the downloaded ZIP file.
6. Enable **PM Auto Outfit Fitter**.

---

## Requirements

* Blender 5.1 or newer
* Windows, Linux, or macOS
